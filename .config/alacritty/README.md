# Alacritty for WSL

## Install Chocolatey
Paste on PowerShell(admin)
```
> choco install alacritty
````
Copy alacritty.yml to
```
C:\Users\"user"\AppData\Roaming\Alacritty\
```

# Alacrity for Linux

```
sudo "package_manager" install alacritty

```
### Or visit https://github.com/alacritty/alacritty
